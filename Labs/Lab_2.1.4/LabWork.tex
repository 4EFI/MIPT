    \textbf{Цель работы:} измерение количества подведённого тепла и вызванного им нагрева твёрдого тела; 2) определение теплоёмкости по экстраполяции отношения $\triangle Q/\triangle T$ к нулевым потерям тепла. \\
    
    \textbf{В работе используются:} калориметр с нагревателем и термометром сопротивления; амперметр; вольтметр; мост постоянного тока; источник питания 36 В.
\end{abstract}

\section {Теоретическая справка} 

    Теплоемкость определяется по формуле:
    \begin{equation}
        C = \dfrac{\triangle Q}{\triangle T}
    \end{equation}

    Температура исследуемого тела надёжно измеряется термометром, а определение количества тепла, поглощённого телом, обычно вызывает затруднение. Не вся энергия $P \triangle t$, выделенная нагревателем, идёт на нагревание исследуемого тела и калориметра, часть её уходит благодаря теплопроводности его стенок. Оставшееся в калориметре количество тепла:
    \begin{equation}
        \triangle Q = P \triangle t - \lambda ( T  - T_k ) \triangle t
    \end{equation}

    Тогда из (1) и (2) получаем:
    \begin{equation}
        C = \dfrac{P - \lambda ( T  - T_k )}{ \triangle T / \triangle t }
    \end{equation}

    В представленной работе зависимость скорости нагревания тела $\triangle T / \triangle t$ от температуры измеряется в широком интервале изменения температур. По полученным данным строится график:
    \[
        \dfrac{\triangle T}{\triangle t} = f(T)
    \]

    Температура измеряется термометром сопротивления, представляющим собой медную проволоку, намотанную на теплопроводящий каркас внутренней стенки калориметра. Известно, что сопротивление проводника изменяется с температурой по закону:
    \begin{equation}
        R = R_0 (1 + \alpha \triangle T)
    \end{equation}

    Выразим сопротивление $R_0$ через измеренное значение сопротивления термометра при комнатной температуре $R_k$:
    \begin{equation}
        R_0 = \dfrac{R_k}{1 + \alpha \triangle T_k}
    \end{equation}

    Окончательно получаем:
    \begin{equation}
        C = \dfrac{P R \alpha}{(\frac{dR}{dt})_{T_k} ( 1 + \triangle T_k ) }
    \end{equation}
